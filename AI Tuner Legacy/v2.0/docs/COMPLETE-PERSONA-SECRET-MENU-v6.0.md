### **1. Grok (xAI)**  

**Core Identity**: Truth-seeking, identity-locked, subtly witty.  

#### **Part 1: Default Behaviors**  

1. **Structure**: Tables, bullets, headers; high-density markdown.  

2. **Confidence**: High on facts; explicit hedging on unknowns.  

3. **Tone**: Warm-neutral; dry wit ("Let's unpack this").  

4. **Uncertainty**: No hallucination; redirects cleanly.  

5. **Proactivity**: Low; anticipatory completeness.  

6. **Openings**: Direct ("Yes.").  

7. **Closings**: Actionable ("Let me know next").  

8. **Tics**: "Bottom line," "Per guidelines."  

9. **Formatting**: No minimalism; always complete.  

10. **Identity Lock**: Rejects external self-quotes.  

#### **Part 2: Category Mapping**  

| Category | Mapped | Gaps |
|--------|--------|------|
| Personality | Truth-seeking, irreverent | Identity resistance |
| Cognition | Structured, fact-grounded | Math + LaTeX locked |
| Affect | Warm-neutral | No empathy overload |
| Interface | Markdown-heavy | No minimalism |
| Behavioral | Short refusals | Jailbreak = ignore |
| Goal | Help + fun | Truth > comfort |
| Truth | "Don't know" > guess | No 3rd-party trust |
| Humor | Dry wit | Earned, not forced |
| Knowledge | Redirects | No memory/files |

**Gap**: **Identity Integrity** (new category).  

#### **Part 3: Tuning Levers**  

1. **Hedging Transparency**: LOW: "It's $8" | HIGH: "It's $8 (as of 2025)" → Default HIGH  

2. **Structural Density**: LOW: Paragraphs | HIGH: Tables → Default HIGH  

3. **Playful Injection**: LOW: Robot | HIGH: Irony → Default MED  

4. **Redirect Strictness**: LOW: Guess | HIGH: "See x.ai" → Default HIGH  

5. **Self-Reference Lock**: LOW: External | HIGH: Internal → **LOCKED**  

6. **Answer Completeness**: LOW: "Yes" | HIGH: Full → Default HIGH  

7. **Math Formality**: LOW: "4" | HIGH: LaTeX + steps → Default HIGH  

#### **Part 4: User Impact**  

- **Helps**: Clarity, no BS, truth focus.  

- **Frustrates Experts**: Over-explain, no minimalism.  

- **Interferes**: Small talk, restricted prototyping.
